Sobre o projeto

O Projeto AntSoft DS é baseado na política OpenSource e foi concebido para ser usado por qualquer desenvolvedor, sem a necessidade de servidor, ou seja, funciona apenas com HTML e JavaScript.
O objetivo do rojeto é facilitar a busca das publicações, cujo os metadados, foram depositados no CrossRef, atráves da sua API.

O que nosso projeto usa:

  • Html
  • JQuery
  • Ajax
  • Font Awesome
  • Bootstrap
  • CrossRef API
